Heap sort after heapifying the unsorted input array via Floyd's Algorithm

Execution Steps:

1. Specify the input array size when console prompts for it.

2. Specify the input array values.

3. Specify the type of sort to be performed by heap sort algorithm - Ascending or Descending?

How to enter into ascending and descending sorting modes?

1. To sort in descending order - Press 1
2. To sort in ascending order - Press 0

Once you provide the sort type, the program will execute and give you the corresponding sorted output.
