BFS and DFS Topological Sort

The program needs four inputs:
1. No. of Nodes
2. No. of Edges
3. Whether node values are characters
4. Edge Pairs

Once the input is given, program executes and computes the BFS and DFS topological sorting order respectively.

It also identifies cycles if any in the given graph.
